Sobre meu aprendizado na Trybe

Neste artigo de forma resumida, quero apresentar o que tenho aprendido aqui na Trybe.

Bloco 1: Unix & Shell, Bash - Foi ensinado a como funciona os hardwares e softwares com o sistema Unix, como a fazer o uso das aplicações nos terminal necessárias para a pessoal desenvolvedora.

Bloco 2: Git & GtHub - Aprendi como funciona e como criar um repositório Git e como conectá-lo remotamente com o GitHub.

Bloco 3: HTML & CSS - Aprendi a como estruturar uma página em HTML, como usar as tags semânticas e como estilizá-la com o CSS.

Aprendizado com qualidade

Nosso currículo é desenhado em parceria com pessoas referência em tecnologia e as empresas mais buscadas para se trabalhar do mercado. Desenvolvemos projetos com problemas da vida real e vivencia os desafios do mercado de trabalho. Com a uma comunidade digital, nós estudantes contamos com momentos de interações para networking, para aprender ensinando e tirando dúvidas.
